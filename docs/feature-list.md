# Функции

## Пользователи:

- Вход в систему
- Создание пользователей администратором
- Отдача списка сотрудников (login, ФИО, email)

## Бронирование:

- Отдача списка доступных комнат/мест
- Создание брони комнат/рабочих мест
- Изменение брони комнат/рабочих мест
- Удаление брони комнат/рабочих мест
- Отдача всех броней всех комнат/мест
- Отдача всех броней по комнате/месту
- Отдача всех броней комнат/мест пользователя
